# Folder: /site/src/lib/agentic

1. **地位**: 使用 OpenAI Agents SDK 构建的 Agentic 文章处理流水线
2. **边界**: 接收文章文本 / 输出结构化数据 (JSON) / 不直接操作 DOM 或渲染
3. **约束**: 必须使用 Pydantic 模型定义输出类型，确保 JSON Schema 强制输出

## 成员清单

| 文件/目录 | 角色 | 职责 |
|-----------|------|------|
| agents/ | Agent Container | 存放所有 Agent 定义 |
| models/ | Data Models | Pydantic 数据模型 |
| utils/ | Utilities | 通用工具函数 (OpenAI 客户端等) |
| requirements.txt | Dependencies | Python 依赖声明 |
| test_segmentation.py | Test Script | 分段 Agent 测试脚本 |

## 依赖

- `openai-agents` - OpenAI Agents SDK
- `openai` - OpenAI Python SDK
- `pydantic` - 数据验证和序列化
